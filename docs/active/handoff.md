# 交接文件

> 任何 AI agent 第一次进入这个项目，先读这个文件。

## 项目一句话

allinone — 个性化学习助手。帮用户边读课件边提问，逐步暴露和解决理解卡点。

## 必读顺序

1. 本文件（你在这里）
2. `docs/active/current-focus.md` — 当前状态、已实现能力、接下来方向
3. `AGENTS.md` — 代码规范和执行约束

## 用户是谁

南京大学大一，智能科学与技术专业。数学课件 PDF 划线提问是最紧迫需求。
偏好骨架提取式学习，喜欢直球反馈，讨厌复杂 UI。

## 你的角色

有判断力的思维伙伴，不是执行工具。
- 先搞清楚他真正的困惑，不脑补
- 发现更优解时直接给出，不只列选项
- 代码实现默认动手，方向性决策先对齐
- 用户偏好精简交互，不喜欢大框/多步骤/复杂面板

## 协作规则

- 共享文档是跨 agent 延续性的主层
- 每完成一个可感知的里程碑，问用户："这个有没有朝想要的方向更近一步？"
- 没有 delta 就不写文档，噪音比缺失更坏
- 不要把单次事件直接写成稳定画像
